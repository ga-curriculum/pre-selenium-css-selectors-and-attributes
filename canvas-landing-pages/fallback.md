## About

This module introduces learners to the fundamentals of CSS selectors and HTML attributes—critical concepts for identifying and interacting with elements during test automation with Selenium. Students will explore how selectors work, including element, class, ID, attribute, and combinator selectors, and how they apply to real HTML structures. The session emphasizes using developer tools to test and refine selectors directly in the browser. The goal is not to teach CSS styling, but to build confidence in targeting elements effectively with selectors that are precise, reliable, and aligned with testing best practices.
